# JT-142 - Q-07 Interview Invitations

JT-142 answers Q-07: "How many interview invitations have I received?"

The implementation adds a deterministic `/metrics/summary` slice that counts `application_events` rows where `event_type = 'interview_scheduled'`.
It does not use an LLM, application status, or cached insight text for the authoritative count.

The dashboard renders an "Interview invitations" card from that endpoint.
The card labels the source as Q-07 and states that it is counted from `interview_scheduled` events.

Verification should include the focused backend metrics API test and the frontend dashboard test.
