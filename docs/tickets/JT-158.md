# JT-158 - Best-Converting Titles

## What Changed

The dashboard now renders a Q-23 "Best-converting titles" section.
It loads deterministic role breakdown rows from `GET /metrics/breakdown?dimension=role`, ranks roles by interview conversion, and displays the interview rate plus supporting numerator and denominator.
The request uses the shared route-backed dashboard filters.

## Why

Q-23 asks which roles get the most interviews.
Existing role breakdown metrics already expose application and interview counts, so this ticket adds the missing dashboard presentation without introducing new metric math or LLM behavior.

## Verification

Run frontend verification from `frontend/`:

```sh
npm run test -- src/pages/DashboardPage.test.tsx
npm run check
```

## Usage Notes

Start the local backend and open the dashboard.
The "Best-converting titles" section ranks role titles by interview conversion for the active dashboard filters.
