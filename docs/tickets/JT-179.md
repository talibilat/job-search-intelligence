## JT-179 - Answer Q-39 adjacent role suggestions

## What changed

`GET /metrics/diagnostics` now includes `adjacent_role_suggestions` for Q-39.
The list contains role segments with interview or offer success, ranked by success rate and sample size.

The dashboard diagnostics card now renders a `Q-39 adjacent role suggestions` section that highlights the strongest role conversion signal.

## Why it changed

Q-39 asks whether there are adjacent roles the user should consider given where they convert best.
The existing role breakdowns could show conversion, but diagnostics did not expose a dedicated adjacent-role answer.

The implementation keeps the answer deterministic by deriving role success signals from local SQLite `applications` and `application_events` data.
It does not use an LLM for counts, rates, or ranking.
