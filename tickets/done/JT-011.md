- Added typed fields for every setting currently documented in `backend/.env.example`.
- Added comma-separated parsing for `JOBTRACKER_CORS_ORIGINS` and `JOBTRACKER_GMAIL_SCOPES`.
- Constrained `JOBTRACKER_DATABASE_URL` to file-backed local SQLite URL schemes.
- Constrained Gmail scopes to the v1 `gmail.readonly` scope.
- Added blank-string normalization for optional `JOBTRACKER_SQLITE_VEC_EXTENSION_PATH`.
- Added user-path expansion for path settings such as `JOBTRACKER_GMAIL_CLIENT_CONFIG_FILE`.
- Added `get_settings()` as a cached dependency helper for later FastAPI dependency injection.
- Added `pydantic-settings` to backend runtime dependencies and updated `uv.lock`.
- Added `backend/tests/test_config.py` covering defaults, `.env` overrides, invalid enum rejection, file-backed local SQLite enforcement, Gmail scope enforcement, `.env.example` parity, and secret-like unknown env value exclusion from settings output.
